Mailgun服务邮件转发实现个性邮箱地址

发表于 2016-02-03   |   分类于 笔记   |  

现在很多域名商都提供邮件转发的功能，之前域名服务器转回国内后当然要找个稳当的备胎来实现了。Mailgun是著名IDC rackspace旗下的公司，和github关系密切，其稳定性毋庸置疑。免费账户有每个月10000封的邮件额度，对普通用户来说简直卓卓有余。这样个性的邮箱地址又可以继续使用了。

注册账户

登录官网注册 https://mailgun.com/signup

Company/Account name 用户/公司名称
Your name 姓名
Your email 邮箱地址
Password 密码
Confirm password 密码确认
Add payment info now (optional) 付费方式，免费用户就不用勾选

激活账户

提交确认之后，登录刚填写的邮箱查收激活邮件，点击Activate Mailgun Account激活。

添加域名

登录已激活的账户，点击Domain菜单，再点击Add New Domain添加将要作为邮件的根域名。

设置DNS解析

按照网站给出的提示，完成下面的操作。

添加TXT记录

1.主机头填@，类型TXT，解析值v=spf1 include:mailgun.org ~all，MX优先级、TTL默认即可。
2.主机头填k1._domainkey（即加粗部分），类型TXT，解析值k=rsa; p=xxx，MX优先级、TTL默认即可。

添加CNAME记录

主机头填email，类型CNAME，解析值mailgun.org，MX优先级、TTL默认即可。

添加MX记录

1.主机头填@，类型MX，解析值mxa.mailgun.org，MX优先级10，TTL默认即可。
2.主机头填@，类型MX，解析值mxb.mailgun.org，MX优先级10，TTL默认即可。

检查DNS记录值

待DNS记录生效后，点击check DNS Records Now检查，若每项记录前都有绿色的沟，该域名就会变成Active状态。

添加路由规则

点击Routes菜单，再点击Create New Route添加路由规则。点击Show Options可直接选择功能方便输入。

Priority 优先级。如填1
Filter Expression 筛选规则。若想接收所有以你域名为后缀的邮件，填catch_all()。而仅想收单一的用户名为前缀的邮件填形如match_recipient("name@domain.com")。
Actions 动作。填写forward("email@email.com")作为转发的目标邮箱地址。
Description 描述。可填可不填。

完成

就这样，酷酷的个人域名就做好了。要注意的是企鹅邮箱好像会屏蔽了mailgun，还是选择大gmail吧。
