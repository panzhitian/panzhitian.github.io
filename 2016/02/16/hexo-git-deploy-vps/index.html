hexo博客用git hooks自动部署到vps

发表于 2016-02-16   |   分类于 笔记 ， hexo   |  

由于文件大小的变化，以及网络的不稳定性，用ftp上传博客不但速度慢，而且操作不可逆。所以用git部署更具优势，也可同时备份到github。这样即使vps有问题也可以从github这边找回一些记录。

如web目录是/home/git/poonchit.im。若目录下有.usr.ini可能会提示非空目录。

设置用户和邮箱

注意：若web目录不是git的话，git hooks脚本会执行出错，所以最好是更改属主为git。如果需要解析php动态页面，可以修改nginx的配置文件,让user为git:git。

配置站点文件

修改_config.yml文件，将会上传到服务器端的hexo.git以及github。
